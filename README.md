## Proceso de Registro y Verificación en Azure ##

El acceso a la plataforma cloud se gestionó bajo el convenio educativo entre la Fundación Universitaria Tecnológico Comfenalco y Microsoft, siguiendo estos pasos de seguridad:

1.Validación Institucional: Se realizó el acceso mediante el portal de Azure for Students utilizando las credenciales institucionales, lo que activa el beneficio de créditos educativos para laboratorios de sistemas.

2.Protocolo MFA (Multi-Factor Authentication): Para garantizar la integridad de la cuenta, se completó un desafío de seguridad mediante MFA. Se optó por la verificación de identidad a través de un código de seguridad enviado vía SMS al dispositivo móvil vinculado.

3.Aprovisionamiento del Tenant: Una vez validado el segundo factor de autenticación, se habilitó el acceso al panel de administración para la creación de recursos y la gestión de despliegues estáticos.
